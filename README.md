## Quick Start

1. Clone the repository:
```bash
git clone https://github.com/Wasim2k5/NeuroMesh.git
cd NeuroMesh
```

2. Start NeuroMesh (automated setup):
```bash
start-neuromesh.bat
```

3. Access the Neural Interface:
```
http://localhost:8080/app/
```
